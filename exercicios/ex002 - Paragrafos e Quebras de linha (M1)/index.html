Paragrafos e Quebras de linha

.html é a extensão de um arquivo de documento criado com a linguagem de marcação HTML (HyperText Markup Language), que é a base para a criação e estruturação de páginas da web.
Esses arquivos contêm o código que define o conteúdo de uma página da web, como textos, imagens, links e vídeos, para que navegadores como Chrome, Firefox e Edge possam exibir a página.
